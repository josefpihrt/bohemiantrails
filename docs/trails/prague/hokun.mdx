# Hokun (Uhříněves - Braník)

## Introduction

Hokun is longest of Prague Trails. It goes through (among others) **Ho**stivař, **Ku**nratice and **N**ovodvorská forest parks.

## Summary

| Property | Value |
| --- | --- |
| Distance | 26/30 km |
| Elevation loss | 120 m |
| Start | 📍 Uhříněves train station <br /> 📍 Kolovraty train station (extended) |
| Finish | 📍 Braník |
