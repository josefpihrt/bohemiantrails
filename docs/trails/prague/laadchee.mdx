# Laadchee (Vysočany - Podbaba)

## Introduction

Laadchee trail offers long descent in the second half with over 170 meters elevation loss.

## Summary

| Property | Value |
| --- | --- |
| Distance | 23/27 km |
| Elevation loss | 30/100 m |
| Start | 📍 Vysočany train station <br /> 📍 Vysočany metro station <br /> 📍 Satalice train station |
| Finish | 📍 Podbaba |
